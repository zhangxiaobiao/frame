# php-48
# 传智播客PHP就业班 48期 博客项目

目录结构
-------------------

```
app                             包含mvc代码文件
    config                      包含项目配置文件
    model                       包含前后台公用的模型文件
    controller                  包含前后台的控制器文件
        frontend                包含前台的控制器文件
        backend                 包含后台的控制器文件
    view                        包含前后台的视图文件
        frontend                包含前台的视图文件
        backend                 包含后台的视图文件
    core                        包含基础类文件
vendor                          包含第三方代码文件
web                             包含静态资源文件
    frontend                    包含前台的静态资源文件
        css                     包含前台的css资源文件
        js                      包含前台的js资源文件
        image                   包含前台图片资源文件
    backend                     包含后台的静态资源文件
        css                     包含后台的css资源文件
        js                      包含后台的js资源文件
        image                   包含后台图片资源文件
    js                          包含前后台公共的js资源文件
    css                         包含前后台公共的css资源文件
    image                       包含前后台公共的图片资源文件
```
